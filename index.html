⚙️ Perlu Setup Dulu

Google Sign-In belum dikonfigurasi. Isi GOOGLE_CLIENT_ID di bagian atas kode (cari === KONFIGURASI ===), lalu buka ulang file ini.

  1. Unduh & hosting file ini di domain sungguhan (GitHub Pages / Netlify / Firebase Hosting) — tidak akan berfungsi di preview Claude.
  2. Buat project di Google Cloud Console → OAuth consent screen → Credentials → OAuth Client ID (Web application) → tambahkan domain hosting kamu di Authorized JavaScript origins.
  3. Salin Client ID ke GOOGLE_CLIENT_ID.
  4. Supaya siapa saja (bukan cuma akun yang kamu daftarkan manual) bisa login, ubah status OAuth consent screen dari Testing ke Production (perlu verifikasi dasar dari Google — bukan proses berat, karena app ini hanya minta scope drive.file).

Tidak perlu bikin/share file Drive manual — setiap orang yang login otomatis mendapat file datanya sendiri di Drive mereka.
